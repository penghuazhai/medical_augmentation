# medical_augmentation
Script for medical image augmentation.

## List of augmentations
* Resize
* RandomFlip
* Inpainting
* OutPainting
* RandomLocalPixelShuffling
* RandomNonlinearTransformation
* RandomShift
* RandomCutout
